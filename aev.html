Animation config
? Here you can make different combinations to find animations to the damage event.
On "Damage origin" it kinda makes Leon rotate to the pivot center.

Known animation types:
00: front damage
01: upper damage
02: stomach damage
03: back damage
06: death
07: launched back
09: cover explosion
!!! For this event type you can make good use of the definition bytes.

Known values:
08 00 02 24

08: activator
00: padding
02: object dependent
24: box ETS index

Use this config to make events remain active only while an enemy or a desctructible object exists. If you change the "02" to "01" the event will be attached to an enemy.
